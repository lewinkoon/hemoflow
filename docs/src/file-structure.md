## File structure

For the script to work properly an structured `files/` directory must be arranged. Each on of the four image series must be contained in a separate directory.

```bash
files/
├───M
│   ├───IM1.DCM
│   ├───IM2.DCM
│   ├───...
│   └───IMX.DCM
├───FH
│   ├───IM1.DCM
│   ├───IM2.DCM
│   ├───...
│   └───IMX.DCM
├───AP
│   ├───IM1.DCM
│   ├───IM2.DCM
│   ├───...
│   └───IMX.DCM
└───RL
    ├───IM1.DCM
    ├───IM2.DCM
    ├───...
    └───IMX.DCM
```
